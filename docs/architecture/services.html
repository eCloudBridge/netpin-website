Microservices Architecture

Detailed documentation of each microservice in the Netpin platform.

Auth Service

Language Go (Gin framework)
Port 8001
Repository services/auth-service

Responsibilities

  • User registration and authentication
  • JWT token generation and validation
  • Password hashing (bcrypt)
  • User profile management
  • API key management

Discovery Service

Language Go (Gin framework)
Port 8000
Repository services/discovery-service

Responsibilities

  • Multi-cloud asset discovery (AWS, Azure, GCP)
  • Kubernetes resource inventory
  • Compliance scanning integration
  • Issue and remediation tracking

Gate Service

Language Go (Gin framework)
Port 8000
Repository services/gate-service

Responsibilities

  • Deployment quality gating
  • Policy evaluation against IDI scores
  • Webhooks for CI/CD integration
  • Vulnerability and code coverage checks

Audit Service

Language Go (Gin framework)
Port 8000
Repository services/audit

Responsibilities

  • Centralized event logging
  • Audit trail recording
  • Deployment history tracking
  • Activity monitoring

Cluster Manager

Language Go (Gin framework)
Port 8002
Repository services/cluster-manager

Responsibilities

  • Cloud provider credential management (encrypted)
  • Cluster discovery (AWS EKS, GCP GKE, Azure AKS)
  • Cluster registration and status tracking
  • Project management

Kubernetes API

Language Go (Gin framework)
Port 8000
Repository services/kubernetes-api

Responsibilities

  • Kubernetes API proxy with authentication
  • EKS token generation (aws-iam-authenticator)
  • Topology data generation
  • Helm release management
  • Resource CRUD operations

Stack Deployment

Language Go (Gin framework)
Port 8000
Repository services/stack-deployment

Responsibilities

  • Helm chart management
  • Application stack definition and deployment
  • Release history and versioning

Graph Service

Language Go (Gin framework)
Port 8000
Repository services/graph

Responsibilities

  • Resource relationship mapping
  • Graph-based queries
  • Topology data enhancement

IDI Service

Language Go (Gin framework)
Port 8003
Repository services/idi-service

Responsibilities

  • IDI score calculation
  • Findings management
  • Trend analysis and forecasting
  • Deployment gate evaluation

Control Panel

Language Go (Gin framework)
Port 8000
Repository services/control-panel

Responsibilities

  • Admin dashboard functionality
  • System configuration management
  • User and role metrics

API Gateway

Language Traefik
Port 80
Repository docker-compose.yml

Responsibilities

  • Request routing to backend services
  • Load balancing
  • SSL termination (prod)
  • Entrypoint management

Frontend

Framework React + Vite
Port 5173 (dev)
Repository frontend

Key Libraries

  • React Query — Server state management
  • Zustand — Client state with persistence
  • React Flow — Topology visualization
  • Recharts — Charts and graphs
  • TailwindCSS — Styling
