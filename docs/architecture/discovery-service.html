Discovery Service

The Discovery Service is the reconnaissance engine of Netpin, responsible for identifying assets, clusters, and resources across multi-cloud environments.

Core Capabilities

  • Multi-Cloud Scanning: Automatically detects EKS, AKS, and GKE clusters.
  • Resource Inventory: Catalogs Nodes, Pods, Services, and Ingresses.
  • Compliance Checks: Runs CIS benchmarks against discovered assets.
  • Drift Detection: Identifies unmanaged resources.

Configuration

The service is configured via environment variables and provider secrets.

Variable Description Default
SCAN_INTERVAL Frequency of discovery scans 300s
ENABLE_COMPLIANCE Run compliance checks during scan true
MAX_CONCURRENT_SCANS Parallel scanning limit 5
