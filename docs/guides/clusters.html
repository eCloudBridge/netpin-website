AWS EKS Clusters

To connect AWS EKS clusters:

Add AWS Cloud Provider

Navigate to Clusters ‚Üí Cloud Providers ‚Üí Add Provider ‚Üí AWS

Enter your AWS Access Key ID and Secret Access Key.

Discover Clusters

Select regions to scan and click "Discover Clusters".

All EKS clusters will be automatically detected.

Connect

Select clusters to connect and assign them to a project.

Required IAM Permissions

{
    "Version": "2012-10-17",
    "Statement": [
        {
            "Effect": "Allow",
            "Action": [
                "eks:ListClusters",
                "eks:DescribeCluster",
                "eks:ListNodegroups",
                "eks:DescribeNodegroup"
            ],
            "Resource": "*"
        }
    ]
}

Manual Cluster Addition

For on-premises or unsupported cloud clusters:

# Get your kubeconfig
cat ~/.kube/config

# Or export from kubectl
kubectl config view --raw > cluster-config.yaml

Then in Netpin UI: Clusters ‚Üí Add Cluster ‚Üí Manual ‚Üí Paste kubeconfig

Managing Clusters

View Cluster Details

Click on a cluster to view:

  • Kubernetes version
  • Node count and status
  • Namespace list
  • IDI score and findings

Edit Cluster

Update display name, project assignment, or refresh credentials.

Delete Cluster

Remove a cluster from Netpin. This does not affect the actual Kubernetes cluster.
