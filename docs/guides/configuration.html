Configuration

Complete reference for configuring Netpin services and components.

Environment Variables

Core Settings

Variable Default Description
JWT_SECRET - Secret for signing JWT tokens (min 32 chars)
JWT_EXPIRY 24h JWT token expiration time
ENCRYPTION_KEY - 32-byte base64-encoded AES key
LOG_LEVEL info Logging level (debug, info, warn, error)

Database Settings

Variable Default Description
DATABASE_URL - PostgreSQL connection string
POSTGRES_HOST postgres PostgreSQL hostname
POSTGRES_PORT 5432 PostgreSQL port
POSTGRES_USER netpin PostgreSQL username
POSTGRES_PASSWORD - PostgreSQL password
POSTGRES_DB netpin PostgreSQL database name

Redis Settings

Variable Default Description
REDIS_URL redis://redis:6379 Redis connection URL
REDIS_PASSWORD - Redis password (if required)

IDI Thresholds

Configure IDI scoring thresholds in the IDI service:

Variable Default Description
IDI_THRESHOLD_CRITICAL 85 Score above which IDI is critical
IDI_THRESHOLD_HIGH 70 Score above which IDI is high
IDI_THRESHOLD_WARNING 50 Score above which IDI is warning
GATE_BLOCK_THRESHOLD 70 IDI score that blocks deployments

Docker Compose Override

Use docker-compose.override.yml for local customizations:

version: "3.8"
services:
  frontend:
    volumes:
      - ./frontend/src:/app/src
    environment:
      - VITE_API_URL=http://localhost:3000
      
  kubernetes-api:
    environment:
      - LOG_LEVEL=debug
